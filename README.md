The repository contains the supplementary material files for the manuscript by Ekaterina Petrova and co-authors 
entitled ‘Dusting off old bones: A new Taymyr mammoth skeleton reshapes woolly mammoth paleobiology’, 
submitted in February 2026 to Quaternary International. The files represent the latest version as of 19 February 2026.
